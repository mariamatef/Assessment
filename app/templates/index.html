Code Challenge Assesssment

This is a basic html with no css, so you can just test the controller easily.

We have 3 APIs in this small controller:

  1. Add a new topic
    • Url: /api/add/topic
  2. Get all articles of a topic
    • Url: /api/fetch/topicName
  3. Status of the system
    • Url: /api/topics
    • Example: http://127.0.0.1:4000/api/fetch?topic=topic2
